# 🛒 E-Commerce Web Application (ShopPro)

## 🚀 Project Overview
A fully functional Amazon-style e-commerce web application built using HTML, CSS, and JavaScript.

---

## ✨ Features
- 🧭 30 Categories with sidebar navigation
- 🛍️ 120 Products (4 per category)
- 🔍 Search functionality
- 🛒 Add to Cart
- ❌ Remove items
- 🔢 Quantity update
- ❤️ Wishlist system
- 🔄 Move Wishlist to Cart
- 💾 Data stored using LocalStorage

---

## 🖼️ Product System
- Realistic product names
- Static image mapping for better performance
- Category-based filtering

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
